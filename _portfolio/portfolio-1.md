## **Journal Papers**

This is an item in your portfolio. It can contain **images, detailed descriptions, or formatted text**. If you name the file `.md`, it will be parsed as **Markdown**. If you name the file `.html`, it will be parsed as **HTML**.

### **How to Structure Your Portfolio Items**
- **Markdown (`.md`) files**: Ideal for text-based content, supporting headings, bullet points, and formatted text.
- **HTML (`.html`) files**: Best for complex layouts, embedding external elements, and interactive designs.
- **Images & Media**: You can embed images using `<img>` tags or Markdown syntax.
